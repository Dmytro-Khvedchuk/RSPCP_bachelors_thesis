## RC2 Section 4: Confronting R5 -- Is Our Data Predictable?

### Service
- `src/app/research/application/rc2_predictability.py` -- `RC2PredictabilityAnalyzer` class
- 7 methods: `build_pe_table`, `build_complexity_entropy_data`, `compare_with_r5`, `build_vr_profile_data`, `build_neff_table`, `build_feasibility_table`, `generate_section4_therefore`
- R5 reference values (arXiv:2502.09079, Table 2, d=5): BTC=0.985, ETH=0.987
- `R5ComparisonResult` value object for delta tracking

### Notebook Cells (14 cells appended to RC2_features_and_profiling.ipynb)
- Section header + R5 context (markdown)
- Compute profiles: PredictabilityAnalyzer + SerialDependenceAnalyzer per (asset, bar_type)
- PE table with RdYlGn_r color gradient (H_norm columns)
- R5 comparison table with delta_h highlighting
- Complexity-entropy plane scatter (d=5), color=bar_type, marker=asset
- PE interpretation markdown
- VR profile multi-line chart for primary bar type
- VR results table with BH-corrected p-values
- VR interpretation markdown
- N_eff + feasibility table with classification coloring
- Feasibility gap horizontal bar chart
- Feasibility interpretation markdown
- Therefore paragraph (generated by service)
- Recommender value proposition argument (conditional vs unconditional predictability)

### Key Design Decisions
- Uses Phase 5 analyzers directly (PredictabilityAnalyzer, SerialDependenceAnalyzer) rather than ProfilingService, to compute only the profiles needed for Section 4
- `_load_bar_data_as_polars()` helper from Section 2 cell is reused (notebook scope)
- Feasibility classification: feasible (gap>0), marginal (gap>-1pp), underpowered (gap<-1pp)
- Bar colors: time=grey, dollar=blue, volume=green, imbalance=orange/coral
- 25 tests covering all methods + edge cases (empty profiles, missing PE, Tier C, unknown assets)

### Thesis Narrative Strategy
- R5's near-Brownian finding is reframed as the **strongest motivation** for the recommender system
- Unconditional PE ≈ 1.0 is expected (efficient markets)
- The value comes from **conditional** prediction: regime, feature, and abstention decisions
- Information-driven bars concentrate observations during high-activity periods where structure exists
- VR tests provide horizon-decomposed predictability (complements unconditional PE)
