University: Національний університет "Львівська політехніка", ННІ ІКНІ.

Key requirements:
- Template: `Шаблон бакалаврської роботи.dotm` (in `thesis/docs/`)
- Plagiarism: >= 70% uniqueness (< 50% = rejected without appeal; < 70% on resubmission = not admitted)
- Defense language: English (application filed)
- English annotation: >= 10 pages (since defense is in English)
- Ukrainian annotation: also required
- Physical submission: hard binding, declaration, supervisor review (відгук), external review (рецензія), plagiarism report, English annotation + defense application
- Supervisor writes review with recommended grade; external reviewer writes рецензія
- Expert commission review must pass before defense
- Template and style compliance checked separately

**Why:** These are non-negotiable procedural requirements. Missing any document or failing plagiarism check blocks defense.

**How to apply:** When writing any section, verify it follows the template format. When advising on content, remind about plagiarism risk in literature review chapter. When planning timeline, account for administrative steps (reviews, binding, checks).
